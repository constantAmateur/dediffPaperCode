This contains the code necessary to perform the analysis and produce the results shown in this paper.  This has been split into four folders
1. cellSignalAnalysis - The source code for the cellSignalAnalysis method as used to generate these results.
2. panCancerAnalysis - The code necessary to prepare data and run the analysis that is the basis of Figure 1 (pan-cancer analysis).
3. liverAndGutAnalysis - The code necessary to perpare data and run the analysis that is the basis of Figure 2 (detailed investigation of Gut and Liver)
4. createFigures - The scripts used to produce the final figures.
